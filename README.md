# P5VJ

A browser-based generative VJ (video jockeying) application built with p5.js -- live, interactive visuals controlled in real time, running entirely in the browser.

## About

Part of a broader creative-technologist practice combining generative art, live visuals performance and front end web design/development. Self-hosted on a Linode VPS.

Live at: [p5vj.com](https://p5vj.com)

## Status

Active development.
